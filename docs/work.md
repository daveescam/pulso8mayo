 Diseño: Biblioteca de Templates Pulso v2 Enhanced (16 templates)
Arquitectura General
Cada template sigue el TEMPLATE_SCHEMA.md completo con:
- aiConfig: provider (moondream/openai/hybrid), fallback, maxRetries
- complianceConfig: tipo normativo (NOM-251, NOM-035, LABOR_LAW), sección, frecuencia, auditabilidad
- pasos[]: Cada paso con tipo, validación, aiVerification, branches, logicRules (con escalation chains, remediation protocols, automated actions)
- completionActions: Notificaciones, status updates, PDF reports, trigger next workflow
